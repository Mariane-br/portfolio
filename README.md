# Portfólio – Mariane Ribeiro Barbosa

Portfólio profissional de **Analista de Qualidade de Software**, com tema escuro, visual inspirado em “Quality Level” e foco em QA, testes e garantia de qualidade. Desenvolvido com Next.js, React e Tailwind CSS.

> **Para o GitHub (campo "About" do repositório):**  
> *Portfólio de Analista de QA — Next.js, React, TypeScript e Tailwind. Fundo interativo, ícones em movimento e seções: Sobre, Experiência, Competências, Formação e Contato.*

---

## Conteúdo

- **Início** – apresentação, cargo, slogan (“Há 5 anos impulsionando qualidade e confiabilidade em soluções de software”) e links (WhatsApp, e-mail, GitHub, LinkedIn)
- **Sobre** – resumo profissional e cards (Processos de QA, Estratégia de Testes, Automação, Métricas)
- **Experiência** – timeline com GTPlan (Analista QA Jr I) e Leucotron (Analista de Qualidade)
- **Competências** – tags com habilidades (Cypress, Postman, JMeter, ERP/S&OP, Scrum, etc.)
- **Formação e Cursos** – graduação, técnico, CTFL (ISTQB), Cypress Udemy, idiomas
- **Contato** – formulário com envio via WhatsApp

## Destaques do layout

- **Fundo** – grade estilo “graph paper” e ícones translúcidos (shield, engrenagem, gráficos, documento, globo, etc.) em todo o site
- **Glow interativo** – brilho em cyan/roxo que segue o cursor na viewport
- **Ícones em movimento** – animação suave de flutuação nos ícones de fundo
- **Design** – tema escuro (#0c1222), gradientes e bordas em cyan

## Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build para produção

```bash
npm run build
npm start
```

## Deploy na Vercel

1. Faça push do projeto para um repositório no GitHub.
2. Em [vercel.com](https://vercel.com), importe o repositório.
3. O Vercel detecta Next.js automaticamente.
4. Deploy.
